# Travel Website | Frontend Project

Proyecto de desarrollo web enfocado en la creación de una landing page para servicios turísticos.

## Descripción
Aplicación frontend diseñada para presentar información sobre destinos y servicios de viaje, con enfoque en experiencia de usuario y diseño visual atractivo.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript

## Funcionalidades
- Visualización de destinos turísticos
- Navegación estructurada
- Diseño responsive para diferentes dispositivos

## UX & Data Considerations
Este tipo de aplicaciones suele implicar interacción con usuarios y potencial recolección de datos, lo cual requiere considerar:
- Formularios de contacto
- Gestión de datos personales
- Transparencia en el uso de la información

## Security & Risk Considerations
Desde una perspectiva de seguridad y cumplimiento, este tipo de plataformas puede presentar riesgos como:
- Recolección de datos sin consentimiento adecuado
- Falta de políticas de privacidad visibles
- Vulnerabilidades en formularios (XSS, inputs no validados)

## Mejores prácticas sugeridas
- Implementación de avisos de privacidad y consentimiento
- Validación de datos de entrada
- Protección de formularios contra ataques comunes
- Minimización en la recolección de datos

## Aprendizajes
Este proyecto permitió fortalecer habilidades en desarrollo frontend y comprender la importancia de integrar consideraciones de seguridad y protección de datos incluso en aplicaciones aparentemente simples.
